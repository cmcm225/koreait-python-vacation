# 크롤링(crawling)
웹에 있는 데이터를 내가 필요한 부분만 빠르게 긁어 오는 기법

## 필요한 라이브러리
- requests: 파이썬에서 웹 통신하게 해줌 
- beautifulsoup4: 데이터를 파이썬객체로 만들어 줌

## 크롤링을 하기 위한 최소한의 HTML지식
1. <어쩌고></어쩌고>: 꺾쇄 -> 태그, 어쩌고 -> 태그 이름
2. <어쩌고>: 열기, </어쩌고>: 닫기
3. <어쩌고>
     <저쩌고></저쩌고>
4. </어쩌고> : 저쩌고 태그가 어쩌고 태그안에 포함되어 있다
5. <어쩌고 뭐시시="아싸">: 
어쩌고 태그가 뭐시기 속성을 가지고 있다
뭐시기 속성의 값은 "아싸"이다
6. id속성값은 중복 불가능, class 속성값은 중복 가능
